# ADB Pull Photos

ADB Pull Photos is a native macOS app for importing photos and videos from Android devices over ADB.

## Highlights

- Native SwiftUI macOS interface.
- Bundled `adb`, so users do not need to install Android platform-tools separately.
- Automatic Android device detection and authorization status.
- Real phone album discovery from common media directories.
- Finder-style media grid with lazy loading for large albums.
- Recent photos and videos appear first.
- Multi-select with single click, Command-click, Shift-click, select loaded, and clear selection.
- Finder folder picker before transfer.
- System-language UI: Simplified Chinese for all Chinese locales, English for all other locales.
- Supports macOS 14.0 and later, including macOS 14.8.

## Installation

Download the DMG from this release, open it, and run `ADB Pull Photos.app`.

If macOS warns that the app is from an unidentified developer, open it from Finder with Control-click, then choose Open.
